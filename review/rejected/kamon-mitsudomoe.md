# 却下: kamon-mitsudomoe(三つ巴)

- **日付**: 2026-06-13
- **点数**: 技術品質で不合格(渦/勾玉の形状が再現できず)
- **理由**:
  - 試行1(3円を中心で接触): 団子状になり巴の渦に見えない
  - 試行2(太いstroke円弧+頭円): リング状になり渦の方向性が弱い
  - 試行3(大円外周+半径R/2の半円弧で勾玉): sweep方向の問題で塗りが潰れ黒円になった
- **根本原因**: 巴(勾玉)は単純な円弧の組み合わせでは形が出ない。頭が大きく尾が細る非対称な渦が必要
- **改善案(次バッチ)**:
  - ベジェ曲線(三次)で勾玉1つを定義し、120度回転コピーで3つ配置
  - または正確な巴の参照座標(頭円・尾の制御点)を実装
  - 完成したら tools/kamon_svg.py の mitsudomoe を有効化し再収録
- **教訓**: 円弧だけで描けない曲線紋(巴・桐・鷹の羽・片喰など)はベジェが必要。テンプレ02の手順に追記済み
